# Job Search Web App

## Description
This project is a job search web application built with a Vite React frontend, Node.js Express backend, and MongoDB database. It aims to provide users with an easy way to search and apply for jobs.

## Tech Stack
- **Frontend**: Vite, React
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/JENIS192/JobSearchingg.git
   cd JobSearchingg
   ```
2. Install dependencies:
   For frontend:
   ```bash
   cd frontend
   npm install
   ```
   For backend:
   ```bash
   cd backend
   npm install
   ```
3. Set up your environment variables in a `.env` file using the provided `.env.example` as a reference.
4. Run the application:
   - For frontend:
   ```bash
   cd frontend
   npm run dev
   ```
   - For backend:
   ```bash
   cd backend
   npm start
   ```
